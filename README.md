# 뮤타블랩스 D반 Python부 자료

##### Python을 위한 가상환경 구축 및 실행:
```bash
python3 -m venv .venv            # 가상환경 만들기
/.venv/Scripts/activate.ps1      # Windows 에서 가상환경 켜기
source /.venv/bin/activate       # Linux, MacOS 에서 가상환경 켜기

pip3 install -r requirements.txt # 파이썬 파일에 필요한 라이브러리 설치

python3 file_name.py             # 가상환경 내에서 파이썬 실행하기

deactivate                       # 다 끝나고 가상환경 나가기
```
